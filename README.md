# ByteBot — O Pet Virtual do Dev

ByteBot é um pet virtual gamificado para desenvolvedores, criado para ser uma forma divertida de visualizar e se manter motivado com o progresso nos estudos e projetos.

A ideia é simples: suas atividades diárias de desenvolvimento, como horas de código, commits e estudos, alimentam seu ByteBot, ajudando-o a evoluir e desbloquear conquistas.

## ✨ Funcionalidades

*   **Evolução:** Comece com um Ovo e veja seu pet evoluir por 6 estágios, de *Sprout* a *DevGod*.
*   **Registro de Atividades:** Insira suas horas de código, commits, projetos e horas de estudo para ganhar XP.
*   **Conquistas:** Desbloqueie conquistas por atingir marcos importantes.
*   **Estatísticas:** Visualize seu progresso com estatísticas e um gráfico de atividade semanal.
*   **Interativo:** Seu ByteBot reage ao seu progresso com diferentes humores e falas.

## 🛠️ Tecnologias

Este projeto foi construído apenas com as tecnologias base da web:

*   HTML5
*   CSS3
*   JavaScript (Puro, sem frameworks ou bibliotecas)

## 💻 Como Usar

Como este é um projeto puramente front-end, não há necessidade de instalação ou build.

1.  Clone este repositório:
    ```bash
    git clone https://github.com/dante908/BYTEBOT-pet-virtual-do-dev.git
    ```
2.  Abra o arquivo `bytebot.html` em seu navegador de preferência.

E pronto! Seu ByteBot está vivo e aguardando seus registros.
